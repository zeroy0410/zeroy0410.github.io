功能描述

  1. 支持注册和登录
  2. 支持简单的分类记账
  3. 支持以饼图的形式展现各个分类占总体收支的数目
  4. 支持增删改数据
  5. 支持以一定的xlsx格式导入导出数据

踩过的坑

  1. 程序开始时数据库需要已经初始化完毕db.create_all()
  2. 使用get方法传参会出未知问题，故全部改为使用url传参
  3. 浮点误差会让数据很奇怪，注意保留两位小数
  4. 不同的数据类型（pandas与string）可能会让数据显示异常（空数据显示为’None’）
